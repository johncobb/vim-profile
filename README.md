## Configure Vim

### Clone vim-profile to home folder
```console
git clone https://github.com/johncobb/vim-profile.git

# copy .vimrc to home filder
cp .vimrc ~/

# install vundle
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

# launch vim
vim

# Install plugins
:PluginInstall
```
### GtG!
